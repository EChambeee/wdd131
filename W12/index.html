A forward knot is one of the easiest knots to make when making a bracelet. It is the same knot twice to make the forward knot. Basically the color you want to be shown is the one that will end up on the right side.

A backward knot is the next easiest knot to make when making a bracelet. It is the same knot twice to make a backward knot.Basically the color you want to be shown is the one that will end up on the left side.

A forward backward knot is a little more difficult to make when making a bracelet. It is like the name says a forward then a backward knot. So the color that you want to be shown will end up on the left.

A backward forward knot is a little more difficult to make when making a bracelet. It is like the name says a backward then a forward knot. So the color that you want to be shown will end up on the right.
