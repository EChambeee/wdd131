A cheveron pattern bracelet is one of the more easier kinds of friendship bracelets to make. It only uses forward and backward knots. The only real difficult thing to do is to arrange the string in the order you want it to go.

This pattern is one that has the normal forwards and backwards knots but also the forward backward and backward forward knot.It would be a little hard to make but not impossible for a beginner.

This pattern uses both the forward backward and backward forward knot. This could be tricky to do because you can get lost in how many times you made the first knot. It would make a cool bracelet if you did this one though.
